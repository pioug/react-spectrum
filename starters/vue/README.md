# Vue Spectrum Starter

Local starter app for validating Vue migration work in this repository.

## Run

```sh
yarn install
yarn dev
```

## Storybook

```sh
yarn storybook
```

Build static Storybook:

```sh
yarn build-storybook
```

Storybook stories follow the React package structure and live in package-level folders like `packages/@vue-spectrum/*/stories`.

## Build

```sh
yarn build
```
