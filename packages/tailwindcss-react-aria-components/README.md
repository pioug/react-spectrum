# tailwindcss-react-aria-components

A Tailwind CSS plugin that makes styling states of React Aria Components easier, with shorter names and autocomplete in your editor. [Read docs](https://react-spectrum.adobe.com/react-aria/styling.html#plugin).

## Migration notice

As of February 19, 2026, this React package is in maintenance mode while Vue migration ports continue.

* Vue replacement package: `tailwindcss-vue-aria-components`.
* Migration guide: [`migration/REACT_TO_VUE_MIGRATION_GUIDE.md`](../../migration/REACT_TO_VUE_MIGRATION_GUIDE.md).
