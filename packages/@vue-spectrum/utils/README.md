# @vue-spectrum/utils

Vue Spectrum utility baseline for class-name composition, media-query helpers, and DOM-ref wrappers.

## Exports

* `classNames`
* `keepSpectrumClassNames`
* `shouldKeepSpectrumClassNames`
* `useMediaQuery`
* `useIsMobileDevice`
* `createDOMRef` / `useDOMRef` / `unwrapDOMRef`
* Re-exports from `@vue-aria/utils`
