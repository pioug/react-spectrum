# @vue-spectrum/s2

Vue Spectrum 2 baseline aggregation package that re-exports foundational Vue Spectrum components and plugin wiring.

## Exports

* `Spectrum2Plugin`
* `Provider`
* `Button`
* `TextField`
* `Checkbox`
* `Radio`
* `RadioGroup`
* `Switch`
* `Dialog`
* `Popover`
* `TableView`
* `TreeView`
