## Events

Use the `@press` event handler to handle interactions via mouse, keyboard, and touch. This is similar to `@click`, but normalized for consistency across browsers, devices, and interaction methods. Read our [blog post](blog/building-a-button-part-1) to learn more.

The `@press-start`, `@press-end`, and `@press-change` events are also emitted as the user interacts with the button. `@press-start` and `@press-end` each receive a <TypeLink links={typesDocs.links} type={typesDocs.exports.PressEvent} />, which provides information about the target and interaction method. See [usePress](usePress) for more details.

```tsx render
"use client";
import {VueButtonExample} from '../../src/VueInterop';
import {useState} from 'react';

function Example() {
  let [pointerType, setPointerType] = useState('');

  return (
    <>
      <VueButtonExample
        onPressStart={e => setPointerType((e as {pointerType: string}).pointerType)}
        onPressEnd={() => setPointerType('')}>
        Press me
      </VueButtonExample>
      <p>{pointerType ? `You are pressing the button with a ${pointerType}!` : 'Ready to be pressed.'}</p>
    </>
  );
}
```

## Pending

Use the `isPending` prop to display a pending state. Pending buttons remain focusable, but are otherwise disabled. Pending state changes are announced to assistive technologies.

```tsx render
"use client";
import {VueButtonExample} from '../../src/VueInterop';
import {useState} from 'react';

function PendingButton() {
  let [isPending, setPending] = useState(false);

  return (
    <VueButtonExample
      isPending={isPending}
      onPress={() => {
        setPending(true);
        setTimeout(() => {
          setPending(false);
        }, 5000);
      }}>
      Save
    </VueButtonExample>
  );
}
```

## Link buttons

The `VueButton` component always represents a button semantically. To create a link that visually looks like a button, use the `VueLink` component instead. You can reuse the same styles you apply to the `VueButton` component on `VueLink`.

```tsx render
"use client";
import {VueLinkButtonDemo} from '../../src/VueInterop';

<VueLinkButtonDemo />
```
